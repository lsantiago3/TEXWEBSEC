# TEXWEBSEC — App Description

## Purpose
Provide small businesses with a fast, affordable, secure way to launch a professional website without technical skills.

## Core Features
1. Secure Website Generator
2. Vendor Dashboard (basic editing)
3. Built-in Security Hardening

## Target Users
- Small business owners
- Local Philly shops
- Veteran-owned businesses
- Non-technical entrepreneurs

## Tech Stack (MVP)
- Frontend: Astro or Next.js
- Backend: Supabase or Firebase
- Security: Middleware + secure headers

## Project Goals
- Publish a secure website in under 10 minutes
- Keep the MVP simple and fast to build
- Provide security-by-default for every site

# Memory Bank

This directory contains documentation and reference materials for the TEXWEBSEC project.

## Contents
- **app-description.md** — Overview of TEXWEBSEC, its purpose, and main features.
- **architecture-decisions.md** — Records of major technical and design decisions.
- **implementation-plans/** — Detailed plans for secure website generation and related modules.

## Purpose
The memory bank acts as a centralized knowledge base for project architecture, planning, and implementation notes.  
It ensures consistency and traceability across all stages of TEXWEBSEC development.

## Assignment Goal
Build a real-world website using proper HTML structure, CSS styling, and JavaScript behavior.

## How TEXWEBSEC Satisfies Every Requirement

### Requirement 1: Semantic HTML Structure

header   -> The sticky navigation bar at the top
nav      -> The links inside the header (Home, AI Builder, SOC, Pricing, Contact)
main     -> All the page content below the header
section  -> Each block: Hero, How It Works, AI Builder, SOC Security, Pricing, Contact
footer   -> The bottom of the page with copyright info
article  -> Individual service cards and pricing cards
form     -> The contact form

### Requirement 2: Separation of Concerns

HTML (structure)     -> defines WHAT is on the page
CSS (presentation)   -> defines HOW it looks
JavaScript (behavior)-> defines WHAT it does when you interact

### Requirement 3: Forms and User Input

input type=text      -> Name field
input type=email     -> Email with built-in browser validation
select               -> Dropdown for service interest
textarea             -> Message box
button type=submit   -> Submit button
JavaScript           -> Form validation on submit

### Requirement 4: Navigation and Links

nav with anchor links (a href=#section-id)
Smooth scroll via CSS scroll-behavior: smooth
Mobile hamburger menu via JavaScript toggle

This is not a fake school project. TEXWEBSEC is a real business:
- Real service: AI website builder + SOC security
- Real clients: 7 businesses from diverse communities
- Real multilingual support: 7 languages
- Real AI integration: ARIA security assistant

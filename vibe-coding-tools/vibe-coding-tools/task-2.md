# Task 2 – My Vibe Coding Tools

These are the tools I selected for my personal Vibe Coding workflow:

1. **GitHub Copilot** – For AI‑assisted coding, explanations, and fast generation.
2. **VS Code** – My main editor for writing, testing, and organizing code.
3. **Git & GitHub** – Version control, repositories, and project management.
4. **Postman** – For testing APIs and understanding request/response flows.
5. **Thunder Client (VS Code extension)** – Lightweight API testing inside the editor.
6. **Node.js + npm** – For running JavaScript projects and installing packages.
7. **Python** – For scripting, automation, and backend practice.
8. **Google Chrome DevTools** – For debugging, inspecting, and testing front‑end code.
9. **MongoDB Atlas** – Cloud database for learning CRUD and backend workflows.
10. **Render / Vercel** – For deploying small projects and testing full‑stack apps.

These tools support my learning style, keep my workflow simple, and help me build projects faster with AI assistance.

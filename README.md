## HTML Concepts Demonstrated (Assignment Checklist)

- DOCTYPE html declaration - Line 1 of every file
- Semantic tags: header, nav, main, section, article, footer - give meaning to each part
- Meta tags inside head - charset (supports all languages), viewport (works on phones), description (Google search text)
- Headings h1 through h3 - page title, section titles, card titles
- Paragraphs p - body text throughout
- Links a href - nav links, call-to-action buttons, smooth scroll anchors
- Lists ul and li - nav menu, feature checklists
- Forms: form, input, select, textarea, button - full contact form with validation
- id attributes - section anchors used by JavaScript for smooth scrolling
- class attributes - hooks for CSS styling
- HTML comments - explain each section clearly in the code
- Separation of concerns - HTML = structure, CSS = style, JavaScript = behavior

## Three Pillars of Web Development

HTML -> STRUCTURE -> What is on the page (the skeleton)
CSS -> STYLE -> How it looks (dark theme, cyan/violet/green colors)
JavaScript -> BEHAVIOR -> What happens when you click (nav, animations, language switch, form submit)

## How to Run This Project in GitHub Codespace

Step 1: Click the green Code button on this repo page
Step 2: Click Open with Codespaces, then New codespace
Step 3: Wait for the Codespace to load (about 30 seconds)
Step 4: In the left file panel, find frontend/index.html and right-click it
Step 5: Click Open with Live Preview
Step 6: Your TEXWEBSEC website opens in a browser tab inside VS Code!
Step 7: Every time you save a file, the preview updates automatically

To get a shareable link: click the Ports tab at the bottom, find the port, click the globe icon.
